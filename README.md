# Yazeed Al-Asmar - Premium Space Portfolio

This is a high-end, modern, and interactive personal portfolio website designed for a Machine Learning Engineer and Data Scientist.

## Features
- **Space Theme:** Dark mode default with a canvas-based interactive particle/starfield.
- **Glassmorphism:** Elegant semi-transparent cards with soft shadows and blurred backgrounds.
- **Dynamic Animations:** Typing cursor effect, floating 3D elements, and scroll-triggered reveals (`IntersectionObserver`).
- **Responsive Layout:** Optimized for Desktop and Mobile viewing using CSS variables and flexbox/grid.
- **Pure JavaScript/HTML/CSS:** Zero framework dependencies for ultra-fast loading and smooth rendering.

## Setup & Deployment
1. Simply open `index.html` in any web browser.
2. The `assets/` folder contains all the images referenced in the website.
3. You can easily zip this entire directory (`index.html`, `style.css`, `script.js`, `assets/`, `README.md`) to send to recruiters. 

## Customization
- **Theme Color:** Open `style.css` and modify `--accent-color` (currently `#EE5926`) to change the global highlight color.
- **Typing Text:** Open `script.js` and edit the `texts` array within the `initTypingEffect` function.
- **Content:** All content (Experiences, Projects, Certifications) is statically declared inside `index.html`. You can duplicate the HTML blocks to add more elements.

Enjoy your new futuristic portfolio!
